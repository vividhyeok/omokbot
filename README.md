# 오목봇

브라우저에서 바로 실행되는 강화학습 교육용 오목봇입니다.
대국이 끝나면 봇이 방금 판을 복습하고, 다음 판에서 더 조심할 자리와 사용자의 반복 흐름을 기억합니다.

## 핵심 기능

- 즉시 시작: 페이지 로드 후 바로 대국 가능
- 학습 흐름: 대국 -> 복습 -> 다음 판 반영 과정을 한눈에 표시
- 개념 툴팁: UI 요소에 마우스를 올리면 강화학습 개념과 연결해 설명
- 선택 설명: 봇이 검토한 후보와 선택 이유를 규칙/기억/경험/최종 선택으로 요약
- 좌표 힌트: 가로 A-O, 세로 1-15 좌표를 표시하고 패널에서도 `8. H` 형식으로 위치 확인
- 봇의 기억: 자주 본 사용자 위치, 위험 자리, 반복 흐름 표시
- 봇 성향: 쉽게 두기, 신중하게, 내 습관 노리기 등 선택 방식 조절
- 보드 색 보기: 보드 위 히트맵으로 규칙/상대 예상/선택/경험 표시
- 승률 추적: 실시간 승률 바와 판별 흐름 차트 제공
- 데이터 관리: 이어하기, 메모리 내보내기/불러오기, 전체 초기화 지원

## 실행 방법

1. 브라우저에서 [index.html](index.html) 열기
2. 시작 화면에서 `게임 시작` 또는 `이어 하기` 선택
3. 하단의 `보드 색 보기` 버튼으로 원하는 모드 선택

별도 서버, 빌드, 패키지 설치가 필요하지 않습니다.

## 조작 가이드

- 돌 두기: 보드 빈칸 클릭
- 다시 하기: 현재 판만 초기화하고 즉시 새 판 시작
- 전체 초기화: 전적/학습/저장 데이터 초기화 후 시작 화면으로 복귀
- 기억 파일 내보내기: 현재 로컬 학습 데이터 백업
- 이어 하기(가져오기): 내보낸 JSON 파일 복원

## 보드 색 보기 모드

- `끔`: 시각화 비활성화
- `규칙`: 오목 규칙상 유리한 자리 강조
- `예상`: 사용자의 다음 수 가능성 강조
- `선택`: 봇이 실제 선택할 확률이 높은 후보 강조
- `경험`: 봇이 지난 판들을 보고 유리하다고 보는 자리 강조

모드는 하단 버튼에서 직접 선택되며, 순환 클릭 없이 바로 전환됩니다.

## 저장 데이터

주요 데이터는 브라우저 로컬 저장소에 보관됩니다.

- 전적/통계: `gomoku-stats`, `gomoku-winprob-history`, `gomoku-loss-history`
- 진행 중 대국: `gomoku-ongoing-*`
- 사용자 패턴 모델: `gomoku-player-model`
- 적응형 모델 가중치: `gomoku-adaptive-weights`
- 실험 설정: `gomoku-experiment-settings`
- 최근 복습 요약: `gomoku-last-learning-summary`

## 개발 노트

이 앱은 AlphaZero류의 완전한 자기대국 엔진을 목표로 하지 않습니다.
브라우저에서 바로 체험할 수 있도록 오목 휴리스틱, 사용자 기억, TensorFlow.js 기반 복습 모델을 섞어 교육용 경험으로 구성합니다.

## 권장 환경

- 최신 Chrome, Edge, Safari, Firefox
- 하드웨어 가속이 켜진 환경 권장

## 문제 해결

- 시작이 느릴 때: 잠시 후 기본 엔진으로 대국 가능 상태가 됩니다.
- 상태가 꼬였을 때: `전체 초기화`를 눌러 런타임 상태까지 초기화하세요.
- 데이터 이전: 내보내기 JSON을 다른 브라우저에서 `이어 하기`로 불러오면 됩니다.
