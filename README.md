# 오목봇

브라우저에서 바로 실행되는 적응형 오목 AI입니다.
사용자가 반복해서 두는 패턴을 누적 학습하고, 다음 수 확률을 시각화해 대국 흐름을 직관적으로 보여줍니다.

## 핵심 기능

- 즉시 시작: 페이지 로드 후 바로 대국 가능
- 적응형 AI: 휴리스틱 엔진 + TensorFlow.js 적응 레이어 결합
- 확률 시각화: 보드 위 히트맵으로 수 선택 가능성 표시
- 승률 추적: 실시간 승률 바와 판별 흐름 차트 제공
- 데이터 관리: 이어하기, 메모리 내보내기/불러오기, 전체 초기화 지원

## 실행 방법

1. 브라우저에서 [index.html](index.html) 열기
2. 시작 화면에서 `게임 시작` 또는 `이어 하기` 선택
3. 하단의 `확률 시각화` 버튼으로 원하는 모드 선택

별도 서버, 빌드, 패키지 설치가 필요하지 않습니다.

## 조작 가이드

- 돌 두기: 보드 빈칸 클릭
- 다시 하기: 현재 판만 초기화하고 즉시 새 판 시작
- 전체 초기화: 전적/학습/저장 데이터 초기화 후 시작 화면으로 복귀
- 기억 파일 내보내기: 현재 로컬 학습 데이터 백업
- 이어 하기(가져오기): 내보낸 JSON 파일 복원

## 확률 시각화 모드

- `끔`: 시각화 비활성화
- `봇 우선`: 봇이 우선 검토하는 자리 강조
- `사용자 예측`: 사용자의 다음 수 가능성 강조
- `봇 선택`: 봇이 실제 선택할 확률이 높은 후보 강조

모드는 하단 버튼에서 직접 선택되며, 순환 클릭 없이 바로 전환됩니다.

## 저장 데이터

주요 데이터는 브라우저 로컬 저장소에 보관됩니다.

- 전적/통계: `gomoku-stats`, `gomoku-winprob-history`, `gomoku-loss-history`
- 진행 중 대국: `gomoku-ongoing-*`
- 사용자 패턴 모델: `gomoku-player-model`
- 적응형 모델 가중치: `gomoku-adaptive-weights`

## 권장 환경

- 최신 Chrome, Edge, Safari, Firefox
- 하드웨어 가속이 켜진 환경 권장

## 문제 해결

- 시작이 느릴 때: 잠시 후 기본 엔진으로 대국 가능 상태가 됩니다.
- 상태가 꼬였을 때: `전체 초기화`를 눌러 런타임 상태까지 초기화하세요.
- 데이터 이전: 내보내기 JSON을 다른 브라우저에서 `이어 하기`로 불러오면 됩니다.
